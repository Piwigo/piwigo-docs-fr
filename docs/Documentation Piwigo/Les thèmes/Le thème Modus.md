# Le thème Modus

## Présentation du thème Modus

Modus est le thème par défaut activé sur Piwigo. 

C’est un thème moderne qui offre une compatibilité garantie avec un maximum de plugins.

Ses caractéristiques :

- thème *responsive* compatible avec toutes les tailles d’écran
- grosses miniatures carrées pour les albums
- barre de menu horizontale
- détection automatique de la taille de photo à afficher
- compatibilité avec les écrans à “haute densité de pixel” (type Retina) grâce à une sélection de taille de photo spécifique
- icônes SVG (icônes toujours nettes quelque soit le niveau de zoom).

![fr-theme-modus.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-04e3510e.png)

## Personnaliser les couleurs de Modus

Modus offre 18 jeux de couleurs différents.

Pour choisir le vôtre, une fois que vous avez activé le thème Modus dans l’administration (Configuration> Thèmes), cliquez sur le bouton Configuration.

![modus-configuration.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-d09fe7d4.png)

Vous pouvez alors voir les différents *skins* ou jeux de couleur disponibles. 

Cliquez sur le bouton “Preview” si vous souhaitez agrandir l’exemple.

Cliquez sur le style de votre choix, puis sur “Enregistrez les paramètres” : vous pouvez aller voir ce que cela donne sur votre galerie !

![modus-styles-fr.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-8f320e16.png)

## Les options de Modus

La configuration du thème Modus offre d’autres options.

![fr-modus-options.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-b3898eb3.png)

### Personnalisation des miniatures des albums

Par défaut, Modus affiche pour les albums des miniatures carrées de 250px de côté.

Vous pouvez décocher l’option “miniatures carrées” : les albums seront alors affichés sous la forme de blocs larges, permettant l’affichage de la description des albums.

![fr-modus-options.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-47d4c146.png)

Vous pouvez également choisir de conserver des miniatures carrées, et modifier la taille de ces miniatures.

### Personnalisation des miniatures des photos

Par défaut, sur une liste de photos, Modus affiche des miniatures en taille XXS.

![fr-modus-miniatures.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-fe7743d1.png)

Vous pouvez si vous le souhaitez modifier la taille des miniatures par défaut.

Cette option peut être personnalisée pour les écrans à haute densité (Rétina)

### Ajouter une bannière

Vous pouvez cocher l’option “Afficher la bannière des pages” sur Modus.

Cela affichera une bannière en haut du menu de votre galerie.

![fr-modus-banniere.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-1defbd80.png)

Le contenu de cette bannière (texte ou image) peut être modifié depuis l’écran Configuration>Options dans l’administration de Piwigo. 

![fr-banniere-config.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-a35faf6b.png)

Pour en savoir plus sur la création d’une bannière pour votre galerie, lisez cet article :
