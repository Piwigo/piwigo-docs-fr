## Qu'est-ce que Piwigo ?

### Présentation éclair de Piwigo

Piwigo est un logiciel de gestion de photothèque en ligne, disponible sur le web et sur application mobile. Il permet d’importer vos photos, vidéos mais aussi d’autres documents (PDF, par exemple), de les organiser, les classer, et les partager de façon publique ou privée.

Piwigo a été conçu à l’origine pour gérer des photos, et est optimisé pour cet usage ; c’est pourquoi, dans cette documentation et dans le logiciel, nous parlons le plus souvent de photos. Mais sachez que ce qui est valable pour les photos est également valable pour tout autre type d’image, et dans la majorité des cas, c’est également valable pour les autres formats de fichiers.

### Pourquoi utiliser Piwigo ?

De nombreuses organisations et des particuliers utilisent Piwigo pour sauvegarder, classer et partager leurs ressources visuelles : photos, vidéos, visuels de communication, brochures...

Les avantages sont nombreux :

- Piwigo est plus fiable et sécurisé que les solutions traditionnelles (serveur de fichiers partagé, disque dur externe, clés USB...) et bien plus adapté au télétravail.
- Piwigo permet de classer ses fichiers de façon souple et intuitive. Ainsi, on perd moins de temps à retrouver les fichiers dont on a besoin.
- Piwigo permet de sécuriser l’accès aux données : on peut facilement définir qui a accès à quoi grâce à une gestion avancée des permissions.
- Piwigo permet de partager un fichier ou un ensemble de fichiers de façon sécurisée et rapide, via un simple partage de lien. Cela évite les envois d’emails avec pièce jointe, les doublons, les risques d’erreur...
- Piwigo permet de créer un véritable site web, public ou privé, pour héberger et présenter vos photos. Cette galerie est personnalisable à vos couleurs.

Pour en savoir plus, [consultez notre site web](http://fr.piwigo.com/).

# Fonctionnement de Piwigo

Piwigo est un logiciel en ligne, accessible depuis un simple navigateur web.

Piwigo est séparé en deux grandes parties :

- L’administration
- La galerie

Ces notions sont importantes pour comprendre la suite de la documentation.

## L’administration

L’administration, c’est le backoffice de Piwigo : c’est l’espace web où l’on peut ajouter des fichiers, créer des albums, organiser, modifier, déplacer et supprimer des éléments, accéder aux options de paramétrage... Seuls les utilisateurs de statut administrateur et webmestre peuvent se connecter à l’administration.

## Les utilisateurs

Lorsque vous créez votre compte sur piwigo.com, vous avez tous les droits : vous avez par défaut le statut Webmestre.

Vous pouvez ensuite créer de nouveaux utilisateurs :

- Les administrateurs : ils auront accès à l’administration de Piwigo pour créer des albums, ajouter des photos...
- Les visiteurs : ils n’auront accès qu’à votre galerie Piwigo pour rechercher, visualiser, télécharger des fichiers.

[En savoir plus sur les utilisateurs](/les-utilisateurs)

## Les albums

Pour ajouter des fichiers sur votre Piwigo, vous devez avant tout créer un album.

Les fichiers sont organisés en albums et sous-albums, un peu comme les dossiers de votre ordinateurs. Il s’agit d’une arborescence infinie.

La particularité, c’est qu’un fichier peut être présent simultanément dans plusieurs albums, sans pour autant créer de doublons !

Vous pouvez utiliser vos albums pour classer vos fichiers dans des dossiers cohérents. C’est également un bon moyen de gérer les droits d’accès de vos utilisateurs.

Vous pouvez en effet décider que certains albums ne sont visibles que par certains utilisateurs. Pour simplifier cette gestion, si vous avez de nombreux utilisateurs, vous pouvez créer des groupes d’utilisateurs : par exemple, un groupe pour chaque service de votre entreprise, ou pour chaque filiale.

[En savoir plus sur les albums](/organiser-les-albums)

## Les tags

En plus des albums, Piwigo vous permet de classer vos photos à l’aide de tags.

Les tags sont des mots-clés que vous pouvez associer aux fichiers. Un fichier peut être associé à un nombre illimités de tags.

Sur votre galerie, les utilisateurs pourront rechercher des fichiers par tag, ou même en combinant plusieurs tags.

Vous pouvez tester la recherche par tag [sur cette galerie de démo](https://demo3.piwigo.com/).

## Les plugins

Pour ajouter des options et des fonctionnalités à Piwigo, vous avez accès à une bibliothèque d’une centaine de plugins.

Un plugin est une extension qui ajoute du code à votre Piwigo. Ce fonctionnement évite de surcharger Piwigo avec des fonctionnalités dont tout le monde n’a pas besoin.

En lisant cette documentation, vous découvrirez de nombreux plugins qui pourraient vous être utiles.

Vous pouvez également contacter le support, si vous êtes client piwigo.com, pour nous présenter vos besoins : nous vous expliquerons quels plugins permettent d’y répondre.

Voici par exemple quelques plugins très populaires :

- **Batch Downloader** : ce plugin permet aux utilisateurs de votre galerie de télécharger une sélection de photos sous la forme d’un ou plusieurs fichiers .zip
- **Download by size** : ce plugin permet aux utilisateurs de votre galerie de choisir la taille de téléchargement d’un fichier
- **User Collections** : ce plugin permet aux utilisateurs de votre galerie de créer des collections / sélections de photos, pour les retrouver facilement ou les partager
- **Advanced Menu Manager** : ce plugin permet de personnaliser les menus de votre galerie et leur apparence
- **Community** : ce plugin permet aux utilisateurs de votre galerie qui ne sont pas administrateurs d’ajouter des fichiers dans votre Piwigo. On parle alors de contributeurs.

Mais il y en a des dizaines et des dizaines d’autres !

# Abonnements Piwigo

Pour utiliser Piwigo, vous pouvez créer un compte sur le site [piwigo.com](http://piwigo.com) : vous profiterez d’une période d’essai de 30 jours pour tester le logiciel.

Ensuite, pour continuer à utiliser Piwigo, vous devrez opter pour une offre d’abonnement.

Nous proposons 4 offres, chacune étant calibrée au mieux pour répondre aux besoins de nos différentes catégories de clients :

- Une offre Pro pour les photographes indépendants et freelances
- Une offre Équipe pour les petites équipes
- Une offre Entreprise pour les entreprises et organisations de taille moyenne
- Une offre VIP pour les plus grandes entreprises et organisations aux besoins les plus exigeants

[Voir les offres](https://fr.piwigo.com/tarifs)

Chaque offre comporte plusieurs limitations :

- nombre d’administrateurs
- espace de stockage

En plus de cela, certaines fonctionnalités et certains plugins sont réservés à certaines offres. Les plugins sont des extensions qui permettent d’ajouter des fonctionnalités à Piwigo.

# Héberger moi-même Piwigo

Piwigo est un logiciel libre et open source. Cela signifie que le code du logiciel est ouvert et accessible à tous. N’importe qui peut télécharger Piwigo et le déployer sur son propre serveur web. Si vous avez des compétences en développement, vous pouvez même apprendre à développer vos propres plugins et ainsi, adapter Piwigo à vos besoins spécifiques.

Si vous avez l’envie et les compétences pour installer, héberger et maintenir vous-même votre Piwigo, vous pouvez télécharger la dernière version sur le site [piwigo.org](http://piwigo.org).

L’utilisation de Piwigo, dans ce cas, sera complètement gratuite.

Si vous faites ce choix, vous devez en comprendre les conséquences :

- Vous n’aurez pas accès au support client, uniquement à la documentation et au support communautaire ;
- Vous serez seul responsable des mises à jour et des sauvegardes ;
- Si vous modifiez les fichiers sources de votre Piwigo, en cas de mise à jour, vos modifications seront “écrasées” ;

Si vous souhaitez héberger vous-même Piwigo tout en bénéficiant du support de notre équipe, vous pouvez opter pour notre [contrat de maintenance](https://fr.piwigo.com/tarifs?plan=self_hosted).

Pour en savoir plus sur l’auto-hébergement, [c’est ici](/hebergez-votre-piwigo).
