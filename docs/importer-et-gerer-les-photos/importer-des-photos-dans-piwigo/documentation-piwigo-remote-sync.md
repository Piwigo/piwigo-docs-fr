# Documentation : Piwigo Remote Sync

← [Importer des photos dans Piwigo](.)

## Piwigo Remote Sync : qu’est-ce c’est ?

Piwigo Remote Sync est un outil de transfert de fichiers pour Piwigo.

Il permet de synchroniser un répertoire de votre ordinateur avec votre galerie Piwigo.

## Piwigo Remote Sync : comment ça marche ?

Le fonctionnement est simple : Piwigo Remote Sync va automatiquement créer les albums selon l'organisation de vos répertoires locaux. Vous pouvez lancer la synchronisation plusieurs fois : Piwigo Remote Sync n'ajoutera que les nouveautés.

## Installer Piwigo Remote Sync

Pour installer Piwigo Remote Sync, il suffit de le télécharger [sur cette page](https://piwigo.org/ext/extension_view.php?eid=851) en cliquant sur le bouton Download.

L’application se présente sous la forme d’un fichier intitulé remotesync-ui.jar. Il suffit de double-cliquer sur ce fichier pour lancer l’application. Piwigo Remote Sync fonctionne sur Windows, Mac et Linux.

Vous avez un problème pour installer ou lancer Piwigo Remote Sync ?

Consultez la FAQ ci-dessous.

### FAQ Piwigo Remote Sync

- **Message d’erreur : Unable to locate Java Runtime**
    
    Pour fonctionner, Piwigo Remote Sync a besoin de Java Runtime, un composant qui permet l’exécution des programmes informatiques développés en Java. [Installez Java Runtime depuis sa page officielle.](https://www.java.com/fr/download/manual.jsp)
    
- **Message d’erreur MacOS : Impossible d’ouvrir car cette app provient d’un développeur non identifié**
    
    Si vous utilisez un ordinateur Mac récent, il est possible que le message “Impossible d’ouvrir car cette app provient d’un développeur non identifié” s’affiche lorsque vous double-cliquez sur le fichier remotesync-ui.jar.
    
    1. Dans le Finder sur votre Mac, repérez le fichier remotesync-ui.jar
        
        ![](https://help.apple.com/assets/63BCA927AAE78C58DD7FBE35/63BCA92EAAE78C58DD7FBE76/fr_FR/058e4af8e726290f491044219d2eee73.png)
        
        N’utilisez pas Launchpad. Launchpad ne vous permet pas d’utiliser le menu contextuel.
        
    2. Cliquez sur remotesync-ui.jar en maintenant la touche Contrôle (Ctrl) enfoncée, puis choisissez Ouvrir dans le menu contextuel.
    3. Cliquez sur Ouvrir.
        
        L’app est enregistrée en tant qu’exception dans vos réglages de sécurité et vous pourrez l’ouvrir par la suite en cliquant deux fois dessus, comme n’importe quelle app enregistrée.
        

## Utiliser Piwigo Remote Sync

Lorsque vous lancez Piwigo Remote Sync, la fenêtre suivante s’ouvre.

![piwigo-remote-sync.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4b3b57e5.png)

Saisissez l’URL de votre galerie, ainsi que votre identifiant et votre mot de passe.

Choisissez ensuite le dossier de votre ordinateur que vous souhaitez synchroniser avec Piwigo en cliquant sur l’icône …

![piwigo-remote-sync.upload.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-1a7d31fb.png)

Cliquez sur Upload : le transfert démarre. Vous voyez défiler à l’écran les photos synchronisées.
