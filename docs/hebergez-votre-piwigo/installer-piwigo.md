---

Si vous rencontrez des difficultés dans l’installation de Piwigo, plusieurs possibilités s’offrent à vous :

- Poster un message sur le [forum communautaire](https://fr.piwigo.org/forum/) avec un maximum d’infos sur votre environnement (version Piwigo, version PHP, système d’exploitation, erreurs rencontrées…)
- Si vous installez Piwigo pour le compte d’une organisation, sachez que l’équipe professionnelle qui développe Piwigo peut offrir un support technique payant pour l’installation et la configuration de Piwigo : [Contactez-nous](https://fr.piwigo.com/contact)
- Enfin, n’oubliez pas que vous pouvez aussi prendre un [abonnement](https://fr.piwigo.com/tarifs) pour héberger Piwigo sur piwigo.com, et ainsi ne pas vous soucier des problèmes techniques 😉
