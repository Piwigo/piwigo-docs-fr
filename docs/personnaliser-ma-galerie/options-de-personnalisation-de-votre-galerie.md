# Options de personnalisation de votre galerie

Il existe différents moyens de personnaliser votre galerie Piwigo :

- En choisissant et en configurant le [thème](/les-themes) de votre choix
- En installant des [plugins](/les-plugins)
- Mais il faut également explorer les différentes options de configuration standard de Piwigo. C’est ce que nous présentons dans cet article.

Pour cela, rendez-vous dans d’administration, Menu Configuration > Options. Ce menu centralise différentes options de personnalisation de votre galerie Piwigo.

## Onglet Général

Le premier onglet est réparti en plusieurs sections.

### Paramètres de base

![fr-config-param-base.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-a6b7caf4.png)

**Modifier le titre de votre galerie**

Ici, vous pouvez choisir le titre de votre galerie : c’est en quelque sorte le nom de votre site web. Il est utilisé à de multiples endroits par Piwigo.

Certains thèmes, comme Bootstrap Darkroom, affichent par défaut le titre de la galerie sur toutes les pages (comme vous pouvez le voir ci-dessous), mais ce n’est pas le cas de Modus, le thème par défaut.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-927326db.png)

**Paramétrer le contenu d’une bannière**

La zone “Bannière des pages” vous permet de créer une bannière qui s’affichera en haut de toutes les pages de votre site comme dans l’exemple ci-dessous.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-c86dde5c.png)

Pour en savoir plus sur la création de bannières pour votre galerie Piwigo, lisez cet article :

[Ajouter une bannière personnalisée à votre galerie](/personnaliser-ma-galerie/ajouter-banniere-personnalisee-piwigo)

**Modifier l’ordre d’affichage des photos**

C’est ici que vous pouvez également modifier l’ordre d’affichage des photos sur les pages. Par défaut, elle s’affichent par ordre d’ajout (du plus récent au plus ancien) puis par ordre alphabétique puis par identifiant, mais vous pouvez personnaliser cet ordre d’affichage.

### Permissions

![fr-config-permissions.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4f39b61c.png)

C’est ici que vous pouvez choisir si vous activez ou non les notes sur votre galerie (option “Permettre les notations”). [En savoir plus sur les notes](/les-commentaires-et-notes/gerer-les-notes-votes)

C’est également ici que vous allez choisir si vous acceptez ou non que les utilisateurs puissent créer un compte en autonomie (option “Permettre l’enregistrement des utilisateurs”). [En savoir plus sur les utilisateurs](/les-utilisateurs/creer-et-gerer-les-utilisateurs)

Enfin, vous pouvez ici autoriser ou pas les utilisateurs à personnaliser leurs préférences d’affichage depuis la galerie (en activant ou pas le menu “Personnaliser”). [En savoir plus sur les préférences](/les-utilisateurs/creer-et-gerer-les-utilisateurs)

### Divers

![fr-config-divers.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4ace566d.png)

Les options disponibles dans la section Divers ne sont pas vraiment relatives à la personnalisation de votre galerie, ce sont plutôt des options générales appliquées à l’ensemble de Piwigo :

- premier jour de la semaine
- historisation des visites
- mise en avant ou pas de l’application mobile
- thème des emails envoyés (clair ou sombre)
- détecter ou pas les doublons lors de l’import d’une photo

## Onglet Afficher

Cet onglet permet de personnaliser votre galerie avec de nombreuses options que vous pouvez activer ou désactiver.

Il est réparti en plusieurs sections.

### Page Principale

![fr-config-page-principale.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4df86627.png)

La section “Page principale” regroupe différentes options pour personnaliser les actions disponibles sur votre galerie.

C’est notamment ici que vous pouvez personnaliser le nombre d’albums par page sur votre galerie. Par défaut, ce nombre est fixé à 12, mais si vous avez beaucoup d’albums, vous pouvez en afficher davantage.

C’est également ici que vous pourrez activer ou désactiver la plupart des icônes disponibles sur la page d’accueil de votre galerie, ou sur la page d’un album.

### Page de la photo

La section “Page de la photo” regroupe les actions et options d’affichage que vous pouvez activer ou désactiver sur la page d’une photo.

![fr-config-page-photo.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-3e496be8.png)

Ainsi, vous pouvez personnaliser les actions disponibles pour une photo, notamment activer ou désactiver le téléchargement.

Vous pouvez également choisir d’activer ou pas la barre de navigation qui permet de faire défiler les photos d’un même album.

Enfin, vous pouvez choisir d’afficher ou pas le menu de la galerie depuis la page d’une photo (par défaut, il est masqué).

### Propriétés de la photo

La section “Propriétés de la photo” permet de choisir les propriétés affichées dans la section des informations sur la page d’une photo.

![fr-config-proprietes.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4d831bde.png)

[En savoir plus sur les propriétés](/importer-et-gerer-les-photos/gerez-les-proprietes-et-metadonnees-de-vos-fichiers)

## Personnaliser l’apparence de votre galerie via les préférences

Il existe une autre possibilité, souvent méconnue, qui permet de modifier certains paramètres d’affichage de votre galerie : ce sont les préférences utilisateurs.

Les préférences permettent de personnaliser par exemple :

- Le nombre de photos par page sur un album
- La période d’affichage via le menu “Photos récentes”
- L’affichage des sous-albums en arborescence dans le menu de navigation
- etc.

Les préférences peuvent être différentes pour chaque utilisateur, mais vous pouvez en tant qu’administrateur :

- Définir les préférences par défaut pour chaque nouvel utilisateur
- Définir les préférences de l’utilisateur “guest” (visiteurs anonymes de votre galerie, non connectés)
- Modifier vos propres préférences et celles des autres utilisateurs.

[En savoir plus sur les préférences](/les-utilisateurs/creer-et-gerer-les-utilisateurs)
