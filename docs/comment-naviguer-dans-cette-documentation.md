# Comment naviguer dans cette documentation ?

**Bienvenue sur la documentation de Piwigo !**

Pour trouver des réponses à vos questions et exploiter au mieux toutes les informations disponibles ici, suivez le guide.

## Utiliser la recherche

Si vous êtes à la recherche d’une information en particulier, utilisez la recherche disponible en haut à gauche du site. 

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-d659b425.png)

Cliquez simplement sur le champ de recherche : vous n’avez plus alors qu’à saisir le texte recherché et tous les contenus de la documentations associés apparaissent.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-c7369c90.png)

## Utiliser le menu du haut

En haut de chaque page, un menu de navigation permet de voir où vous êtes dans la documentation, et de naviguer dans l’arborescence.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-0a124d58.png)

## Utiliser le menu de gauche

Le menu de gauche, présent sur toutes les pages, liste toutes les sections de la documentation : import des photos, navigation sur la galerie, gestion des albums…

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-6e06fb9a.png)

Utilisez la barre de défilement ou la roulette de votre souris pour naviguer dans le menu de gauche et visualiser toutes les sections disponibles.

Cliquez sur une section pour ouvrir le menu déroulant avec tous les articles de cette section.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-a2ef3394.png)

Pour chaque section, une page Sommaire liste tous les articles de la section.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-097bd137.png)

## Naviguer au sein d’un article

A droite de chaque article, le sommaire vous permet de visualiser les principaux sujets présents dans l’article et de cliquer sur celui qui vous intéresse pour accéder directement à son contenu.

![Sommaire d’un article](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-b39cf69e.png)

Sommaire d’un article

## Activer le mode sombre

Pour activer ou désactiver le mode sombre, cliquez sur l’icône représentant un soleil en haut à droite de chaque page.
