# Créer des collections (plugin User Collections)

## Qu’est-ce que le plugin User Collections ?

Le plugin User collections permet aux utilisateurs (administrateurs ou non) connectés à votre galerie de créer des sélections de fichiers, pour les retrouver facilement, ou pour les partager (y compris à des personnes qui ne disposent pas de compte sur votre galerie Piwigo).

Chaque utilisateur peut créer un nombre illimité de collections.

Pour utiliser cette fonctionnalité, vous devez activer le plugin **User Collections**.

!!! Warning "Attention"
    Si vous êtes client d’une offre piwigo.com, ce plugin n’est accessible qu’à partir de l’offre Entreprise.


## Définir les autorisations associées aux collections

Une fois le plugin User Collections activé, vous devez vous rendre dans la configuration du plugin en cliquant sur “Configuration”.

L’onglet Configuration permet de définir les droits des utilisateurs :

- Est-ce qu’ils ont le droit de rendre leur collections publiques (c’est à dire, accessible via un lien de partage, y compris à une personne n’ayant de pas de compte sur votre galerie)
- Est-ce qu’ils peuvent envoyer leur collections publiques par email
- Est-ce qu’ils peuvent envoyer leurs collections à un administrateur

## Comment créer des collections ?

Une fois le plugin activé, les utilisateurs connectés peuvent ajouter des photos à une collection :

- Depuis la page Album, en passant la souris sur la vignette d’une photo
    
    ![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-e6e30c12.png)
    
- Depuis la page d’une photo, grâce à l’icône en forme d’étoile située dans la barre d’outils
    
    ![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-5fd6b90a.png)
    

Il est possible d’ajouter une photo à une collection existante, ou bien d’en créer une dans la foulée.

Toutes les collections créées par un utilisateur sont visibles depuis le menu Collections. Pour chaque collection, on affiche le nombre de photos qu’elle contient.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4416c572.png)

La page Collections liste toutes les collections créées par l’utilisateur, permet de les ouvrir (pour les partager par exemple) ou de les supprimer.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-0fbbc131.png)

## Comment partager une collection ?

Lorsque l’on est sur la page d’une collection, plusieurs icônes spécifiques à la collection sont disponibles dans la barre d’outils (en fonction des autorisations définies dans la configuration du plugin).

### **Partager un lien vers la collection**

![collection.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-383da7f0.png)

L’icône de partage ouvre une fenêtre qui permet de générer un lien de partage sécurisé. Vous pouvez, si vous le souhaitez, personnaliser la fin de l’URL ou conserver l’URL générée par défaut.

Vous pouvez si vous le souhaitez, définir un mot de passe pour accéder à cette collection, ainsi qu’une date d’expiration.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-15a1b5c6.png)

Une fois le lien de partage créé, une fenêtre vous permet de récupérer le lien généré pour le partager.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-5edf95d2.png)

Le lien généré permet de visualiser uniquement la collection : il ne donne pas accès au contenu des albums privés.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-d0bbe20e.png)

### Envoyer une collection par email

![collection-mail.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-ca8f3da8.png)

L’icône “email” ouvre une fenêtre qui permet de partager la collection directement par email.

En fonction des droits définis dans la configuration du plugin, vous pouvez envoyer une sélection par email à une personne extérieure, à un administrateur, ou aux deux.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-756ec59b.png)

Le mail est automatiquement envoyé au destinataire.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-a11a46b3.png)

## Gérer une collection sur la galerie

![collection - copie 2.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-4f3093be.png)

Sur la page d’une collection, l’icône en forme de crayon permet de modifier son nom et d’ajouter une description.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-937340a0.png)

![collection - copie 3.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-67372f9a.png)

L’icône en forme de balai permet de vider le contenu d’une collection.

![collection - copie 4.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-34926fec.png)

L’icône en forme de corbeille permet de supprimer la collection.

## Gérer les collections dans l’administration

Les administrateurs peuvent avoir accès aux collections créées depuis la configuration du plugin User Collections.

Un moteur de recherche permet de filtrer les collections par nom et par utilisateur.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-d25f9c33.png)

Depuis cet écran, l’administrateur peut retrouver les collection créées, les modifier, les exporter au format CSV, et les supprimer.
