# Votre galerie : présentation

## Préambule

Votre galerie Piwigo est le site web qui présente votre photothèque.

La galerie Piwigo est utilisée par les personnes qui ont besoin de rechercher, visualiser, télécharger du contenu sur votre photothèque.

Votre galerie est séparée de l’administration, qui permet d’importer des photos, de créer et gérer des albums, des tags, des utilisateurs, de modifier les contenus et d’administrer votre galerie (thème, plugins, langues, paramétrage…).

Votre galerie Piwigo peut être :

- publique : n’importe quel visiteur du site peut voir tous les contenus, même sans être connecté
- privée : seul les utilisateurs connectés ont le droit de visualiser les contenus sur la galerie
- ou mixte : sur une même galerie, certains albums peuvent être publics et d’autres privés.

Si vous n’êtes pas administrateur, vous n’avez accès qu’à la galerie.

Si vous êtes administrateur, vous avez accès à l’administration ET à la galerie.

## Page d’accueil de votre galerie

Chaque galerie est différente ! Toutefois, lorsque l’on installe Piwigo ou quand on crée un compte sur le site Piwigo Cloud, on commence tous avec la même galerie, vide de tout contenu.

Cette galerie utilise le thème [Modus](../les-themes/le-theme-modus) et les paramètres par défaut de Piwigo. C’est cette version “par défaut” que nous allons présenter ici. Mais comme vous le verrez plus tard, il existe de nombreux moyens de personnaliser votre galerie.

La page d’accueil de votre galerie comporte les éléments suivants :

- Le menu de navigation qui contient un champ de recherche et un lien de connexion
- Une liste d’albums (si vous avez créé au moins un album) et une barre d’outils
- Un pied de page avec un texte par défaut. Pour les galeries hébergées sur Piwigo Cloud, vous y trouverez également un lien de connexion.

![accueil.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-1b1f4c8b.png)

Le menu de navigation vous donne accès à différents moyens d’accéder aux contenus de votre galerie :

- Naviguer dans les albums
- Voir les photos les plus vues, les plus récentes, ou les mieux notées
- Voir les photos par date d’ajout (calendrier)
- Naviguer dans les tags
- Accéder à la recherche avancée
- …

[En savoir plus sur le menu de navigation](../personnaliser-ma-galerie/personnaliser-le-menu-de-votre-galerie)

Sur la page d’accueil, les icônes de la barre d’outils donnent accès par défaut à 3 actions :

- Visualiser toutes les photos (tous albums confondus).
- Visualiser les photos par date d’ajout
- Visualiser les photos par date de prise de vue

À partir du moment où vous choisirez l’une de ces options, vous accéderez à la vue “Liste” qui donne accès à d’autres actions. Cette vue Liste est décrite dans la [documentation de la Page Album](les-albums-sur-votre-galerie).

Si vous n’êtes pas connecté, vous voyez ce que n’importe quel visiteur de votre galerie pourra voir, c’est à dire les albums définis comme publics.

Si vous vous connectez, vous accéderez aux albums privés auxquels vous avez accès, ainsi qu’à d’autres fonctionnalités réservées aux utilisateurs connectés, comme ajouter une photo à ses favoris, ou personnaliser ses préférences.

## Accéder à la galerie depuis l’administration

Même si vous êtes administrateur, vous utilisez fréquemment votre galerie.

C’est en effet la façon la plus simple de visualiser et rechercher des fichiers présents dans Piwigo.

Vous avez également besoin de visualiser l’impact des modifications que vous effectuez dans l’administration.

Il existe plusieurs moyens d’accéder à votre galerie directement depuis l’administration de Piwigo.

### Accéder à l’accueil

Dans l’administration de Piwigo, à tout moment vous pouvez basculer sur votre galerie en cliquant sur le bouton Visiter en haut à gauche.

![visiter.png](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-2c17f8fe.png)

### Accéder à un album

Depuis la liste des albums dans l’administration, vous pouvez accéder directement à un album dans la galerie en cliquant sur l’icône représentant un oeil.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-2b83e99b.png)

Lorsque vous éditez un album dans l’administration, vous pouvez accéder directement à cet album dans la galerie en cliquant sur “Ouvrir dans la galerie”.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-e16dab89.png)

### Accéder à une photo

Lorsque vous éditez une photo dans l’administration, vous pouvez accéder directement à cette photo dans la galerie en passant la souris sur la miniature et en cliquant sur le bouton “Ouvrir dans la galerie”.

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-a0144b63.png)

![Untitled](https://ressources.piwigo.com/uploads/c/v/7/cv7jpz6hf8/2026/05/25/20260525140116-7b813d1a.png)

## Personnaliser votre galerie

Il existe de nombreux moyens de personnaliser l’apparence de votre galerie et ses fonctionnalités.

Si vous avez le statut webmaster sur Piwigo, vous avez accès à toutes ces options.

Il est important de les connaître pour offrir aux utilisateurs de votre galerie la meilleure expérience possible.

Dans les chapitres suivants, nous présenterons, pour les principales pages, les options à votre disposition.

Pour obtenir davantage d’informations sur la personnalisation de l’apparence de votre galerie, visitez les chapitres suivants :

[Les thèmes](../les-themes)

[Personnaliser ma galerie](../personnaliser-ma-galerie)
